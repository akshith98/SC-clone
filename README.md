## About The Project



This is a SC.exe clone. It is a command line program used for
communicating with the Service Control Manager and services.


### Built With

* [C++]()

## Getting Started

To get a local copy up and running follow these simple steps.

### Prerequisites

g++ installed or Visual Studio with C++ extention 

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/akshith98/SC-clone.git
   ```
